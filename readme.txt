Instructions for using the 'print pdfs directly out of file explorer':

----------------------------------------------------------------------------------------------------------------------------------------------

Ways of print your files:
    (1) Optionsmenu: Choose File or Folder via Filedialog
    (2) Drag and Drop: Open File Explorer and just drag selected Files into right (left for Folders; right for files) file

Points of Problems:
    - all Files have to be pdf's
    - if u select a folder, all files have to be pdf's
